Login imported users from Joomla 2.5.9 to WordPress 4.1

== Description ==

Login migrated users from Joomla to WordPress
Tested on WordPress 4.1 with users migrated from Joomla 2.5.9
After first successful login, password field will be automatically hashed by WordPress.
In case it does not work, just deactivate.

== Installation ==

1. Upload the entire contents of the zip file to your plugin directory
2. Activate the plugin through the 'Plugins' menu in WordPress
